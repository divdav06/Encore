# Selma Sport

Version Next.js prête pour Vercel.

## Lancer en local

```bash
npm install
npm run dev
```

## Déploiement sur Vercel

- Importer le dépôt GitHub dans Vercel
- Framework détecté : Next.js
- Optionnel : ajouter la variable d'environnement `API_FOOTBALL_KEY`

## API intégrée

- `/api/matches` retourne les données du tableau de bord

Sans clé API, le site utilise des données de démonstration.
